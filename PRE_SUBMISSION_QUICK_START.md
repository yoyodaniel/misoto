# Quick Start: App Store Submission

## 🚀 Fast Track Checklist (Essential Items Only)

### 1. App Store Connect Setup (30 minutes)
- [ ] Create app listing in App Store Connect
- [ ] Upload app icon (1024x1024)
- [ ] Add app description (use `APP_STORE_PROMPT.md` for reference)
- [ ] Add keywords
- [ ] Set category: Food & Drink
- [ ] Set price: Free

### 2. Screenshots (1 hour)
- [ ] Take screenshots on iPhone (at least 3-5 showing key features)
- [ ] Upload to App Store Connect for required screen sizes
- [ ] Screenshots must be actual app screenshots (not mockups)

### 3. Subscriptions Setup (30 minutes)
- [ ] Create subscription group: "Misoto Premium"
- [ ] Add monthly subscription: `com.misoto.premium.monthly` - $4.99
- [ ] Add yearly subscription: `com.misoto.premium.yearly` - $49.99
- [ ] Verify Product IDs match code exactly

### 4. Legal Requirements (15 minutes)
- [ ] Host Privacy Policy on website (URL: _______________)
- [ ] Host Terms of Service on website (URL: _______________)
- [ ] Add Privacy Policy URL to App Store Connect
- [ ] Add Terms of Service URL to App Store Connect
- [ ] Complete App Privacy questionnaire in App Store Connect

### 5. Build & Upload (30 minutes)
- [ ] Clean build in Xcode (⌘+Shift+K)
- [ ] Archive build (Product → Archive)
- [ ] Validate archive
- [ ] Upload to App Store Connect
- [ ] Wait for processing (10-30 minutes)

### 6. Final Submission (15 minutes)
- [ ] Select build in App Store Connect
- [ ] Complete version information
- [ ] Add "What's New" text
- [ ] Review all information
- [ ] Submit for Review

**Total Time**: ~3 hours
